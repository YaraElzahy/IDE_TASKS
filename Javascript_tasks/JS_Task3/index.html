Question 8:
Which of these alerts are going to execute?
What will the results of the expressions be inside if(...)?
line 1: if (-1 || 0) alert( 'first' );
line 2: if (-1 && 0) alert( 'second' );
line 3: if (null || -1 && 1) alert( 'third' );
Answer:


line 1: executes first, -1 || 0 = -1 (true), and thus 'first' appears in alerts
line 2: doesn't execute, -1 && 0 = 0 (false)
line 3: executes second, && has higher priority than ||, and thus -1 && 1 = 1, then null || 1 = 1 (true)
